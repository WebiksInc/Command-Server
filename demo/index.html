GET '/raw?path=...?args=...'

path = path to executable file

args = arguments for this file (as string)

  • /raw?path=notepad.exe&args=/A C:\Users\Public\example.txt

GET '/run/name?arg1=...?arg2=...?arg3=...'

name = name of action

arg_i = arguments for this action (as implement in server)

  • /run/openNotepad?option=A&path=C:\Users\Public\example.txt

  • /run/openFileExplorer?directory_path=C:\Users
